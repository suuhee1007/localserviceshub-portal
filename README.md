# LocalServicesHub Portal

React frontend for the LocalServicesHub application.

## Run locally

1. Install dependencies:

   ```bash
   cd localserviceshub-portal
   npm install
   ```

2. Start the app:

   ```bash
   npm run dev
   ```

The portal will be available at `http://localhost:3000` and proxies API calls to `http://localhost:8080/api`.
